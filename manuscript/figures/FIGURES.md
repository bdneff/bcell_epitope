# Figure provenance

Every figure in the log is regenerated by a named script from tracked inputs. One entry per figure.

| Figure | Script | Inputs | Command |
|--------|--------|--------|---------|
| `hel_crossab_map.png` | `analysis/plot_hel_crossab_map.py` *(to write)* | `benchmark/antigen_alanine_scanning_benchmark_v2.xlsx` | `python analysis/plot_hel_crossab_map.py` |
| `lysozyme_structure.png` | *(to render — VMD/PyMOL of apo 1AKI w/ label residues)* | `md/1AKI/apo/out/md_ref.pdb` | — |
| `hpr_structure.png` | *(to render — 2JEL chain P)* | `structures/HPr_2JEL.pdb` | — |
| `vegf_structure.png` | *(to render — 1BJ1 V,W dimer)* | `structures/VEGF_1BJ1.pdb` | — |
| `mtsp1_structure.png` | *(to render — 3NPS chain A)* | `structures/MTSP1_3NPS.pdb` *(pending loop model)* | — |
| `bonta1_structure.png` | *(to render — 2NYY Hc)* | `structures/BontA1-Hc_2NYY.pdb` | — |

## Generated tables

| Table | Script | Inputs | Command |
|-------|--------|--------|---------|
| `sections/tables/<antigen>.tex` (per-antigen ΔΔG_bind, one column/antibody) | `analysis/make_label_tables.py` | `benchmark/antigen_alanine_scan_extracted_SIMPLE_v1.csv` | `python3 analysis/make_label_tables.py` |

Convention: a figure referenced in the log but not yet regenerated shows a placeholder box
(`\figorbox`), so `main.tex` always compiles. Add the row here the moment the figure is named.
