# Figure provenance

Every figure in the log is regenerated by a named script from tracked inputs. One entry per figure.

| Figure | Script | Inputs | Command |
|--------|--------|--------|---------|
| `hel_crossab_map.png` | `analysis/plot_hel_crossab_map.py` *(to write)* | `benchmark/antigen_alanine_scanning_benchmark_v2.xlsx` | `python analysis/plot_hel_crossab_map.py` |
| `<key>_structure.png` (plain cartoon, key ∈ hel_apo/lysozyme/hpr/vegf/mtsp1/bonta1/ifngr/tf/hcmvgb/hgh; `hel_apo` = apo HEL starting structure from `structures/1AKI.pdb`) | `analysis/render_structures.sh` → `analysis/vmd/render_structure.tcl` | each run's `md/<PDB>/apo/out/md_ref.pdb` or input structure | `analysis/render_structures.sh` (local; needs VMD) |
| `<key>_<pdb>_ddg.png` (per-antibody, residues colored by ΔΔG) | `analysis/render_ddg_figures.sh` → `analysis/vmd/render_ddg.tcl` | `manuscript/figures/ddg/<key>_<pdb>.dat`, input `structures/*.pdb` | `python3 analysis/make_ddg_coloring.py && analysis/render_ddg_figures.sh` |
| `ddgbar.png` (single shared colorbar, global ±max\|ΔΔG\| scale, used by all antigen figures) | `analysis/make_ddg_coloring.py` (matplotlib) | benchmark CSV | `python3 analysis/make_ddg_coloring.py` |
| `sasa_baseline.png` (static-exposure baseline: every residue, rel. SASA vs ΔΔG, exposure decision line) | `analysis/compute_sasa.py` → `analysis/plot_feature_vs_label.py` (bcell-repair env) | `benchmark/features/sasa_apo_singleframe.csv` + benchmark CSV | `micromamba run -n bcell-repair python analysis/compute_sasa.py && ... plot_feature_vs_label.py --feature-csv benchmark/features/sasa_apo_singleframe.csv --feature-col rel_sasa --label "rel. SASA (apo)" --feature-threshold 0.25` then `cp analysis/figures/sasa_allres_pooled.png manuscript/figures/sasa_baseline.png` |
| `bfactor_baseline.png` (static-mobility baseline: per-residue crystal B-factor z vs ΔΔG; hotspots are rigid) | `analysis/compute_bfactor.py` → `analysis/plot_feature_vs_label.py` (bcell-repair env) | `benchmark/features/bfactor_crystal.csv` + benchmark CSV | `micromamba run -n bcell-repair python analysis/compute_bfactor.py && ... plot_feature_vs_label.py --feature-csv benchmark/features/bfactor_crystal.csv --feature-col bfactor_z --label "crystal B-factor (z, per structure)" --feature-name "B-factor" --feature-threshold 0 --xdir "more mobile" --title "If crystallographic mobility were the epitope label" --outprefix bfactor_allres` then `cp analysis/figures/bfactor_allres_pooled.png manuscript/figures/bfactor_baseline.png` |

## Generated tables

| Table | Script | Inputs | Command |
|-------|--------|--------|---------|
| `sections/tables/<antigen>.tex` (per-antigen ΔΔG_bind, one column/antibody) | `analysis/make_label_tables.py` | `benchmark/antigen_alanine_scan_extracted_SIMPLE_v3.csv` | `python3 analysis/make_label_tables.py` |
| `sections/tables/binary_survey.tex` (Tier-2 binary critical-residue antigen survey) | `analysis/make_binary_label_table.py` | `benchmark/iedb_shotgun_candidates_v1.csv` | `python3 analysis/make_binary_label_table.py` |
| `benchmark/iedb_shotgun_candidates_v1.csv` (IEDB shotgun candidate antigens) | `analysis/fetch_iedb_shotgun.py` | IEDB query API (live) | `python3 analysis/fetch_iedb_shotgun.py` |
| `<key>_{binary,freq}_tier2.png` + `<key>_freqbar.png` (Tier-2 critical-residue maps; key ∈ dengueE/ha) | `analysis/make_tier2_coloring.py` → `analysis/vmd/render_ddg.tcl` | IEDB query API + `structures/{DengueE_1OKE,HA_1HGG}.pdb` | `python3 analysis/make_tier2_coloring.py` then render manifest `figures/ddg/tier2_manifest.tsv` |
| `benchmark/tier2_labels_<key>.csv` (WT-verified critical residues + antibody counts) | `analysis/make_tier2_coloring.py` | IEDB query API + structure | `python3 analysis/make_tier2_coloring.py` |

Data: `benchmark/antigen_alanine_scan_extracted_SIMPLE_v3.csv` = v1 + SKEMPI-2.0-derived antigen-side
alanine rows for IFN-γ receptor (1JRH), Tissue factor (1AHW), HCMV gB (5C6T) + hGH PDB handle (1HGU).
New rows provenance: `benchmark/antigen_alanine_scan_SKEMPI2_new_v1.csv` (built from SKEMPI 2.0, BSC mirror;
ddG=RT·ln(Kd_mut/Kd_wt), replicates averaged; PDB numbering via `Mutation(s)_PDB`).

Convention: a figure referenced in the log but not yet regenerated shows a placeholder box
(`\figorbox`), so `main.tex` always compiles. Add the row here the moment the figure is named.
