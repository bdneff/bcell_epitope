\noindent
Running record of the project: predict B-cell epitope residues from \emph{antigen-only}
information, using alanine-scan $\Delta\Delta G_\mathrm{bind}$ as residue-level binding-importance
labels and asking whether antigen dynamics / solvation / energetics carry predictive signal
\emph{beyond static surface exposure}. Read in order; each section is one iteration --- what we
tried, what we found, what it changed. Not a formal manuscript. Every figure is regenerated by a
named script (recorded in \texttt{figures/FIGURES.md}); every number traces to a file or command;
unconfirmed values are flagged \texttt{[verify]}.
